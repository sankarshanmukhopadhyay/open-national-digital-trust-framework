# ONDTF v0.1.0 — Initial Public Draft

This release establishes the Open National Digital Trust Framework as a jurisdiction-neutral reference framework with country and sector profiles.

The seed defines the framework's design objective, principles, architecture map, governance and assurance structure, adoption method, repository authority boundaries, and initial India-profile skeleton. It also establishes the relationship with the Trust Systems Meta-Model and Trust Infrastructure Schemas without collapsing those repositories into ONDTF.

Version 0.1.0 is suitable for architecture review, terminology review, issue creation, and work-package planning. It is not yet a stable normative standard or production conformance baseline.

## Next milestone

Version 0.2.0 will perform formal TSMM/TIS alignment, produce a traceability matrix and information model, and define the first normative conformance targets.
