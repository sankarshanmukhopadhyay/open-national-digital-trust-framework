# Jurisdiction Profile Path

**Audience:** National programme teams, legal-policy teams, and profile maintainers  
**Prerequisites:** [ONDTF in One Hour](one-hour.md), or equivalent familiarity  
**Estimated time:** 2–3 hours  
**Outcome:** a role-specific understanding of the framework, with enough context to use the reference documentation confidently.

## Recommended sequence

| Order | Read | Why it comes here |
|---:|---|---|
| 1 | [Framework independence](../foundations/framework-independence.md) | Understand what remains stable across jurisdictions. |
| 2 | [Scope and non-goals](../core-specification/scope-and-non-goals.md) | Preserve the framework boundary. |
| 3 | [Jurisdiction profiles](../../profiles/index.md) | Understand the profiling method. |
| 4 | [Governance framework](../governance/index.md) | Map competent authorities and decision rights. |
| 5 | [Standards selection](../standards/index.md) | Choose standards without hard-wiring the core. |
| 6 | [Assurance levels](../assurance/levels.md) | Specialise assurance carefully. |
| 7 | [International federation](../international/index.md) | Plan cross-border recognition. |
| 8 | [India profile](../../profiles/india/index.md) | Review the current illustrative profile. |
| 9 | [Conformance](../implementation/conformance.md) | Define profile and implementation claims. |

## Reading method

Read the pages in order on the first pass. On later visits, use the sidebar and search as a reference system. The sequence is curated; it does not imply that later pages are normatively subordinate to earlier ones.
