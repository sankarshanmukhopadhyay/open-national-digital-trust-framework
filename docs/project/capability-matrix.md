# v0.5.0 capability matrix

The matrix records why v0.5.0 is described as a **Feature Complete Draft**. “Complete” means the planned domain is represented and integrated at framework level. It does not imply production certification or independent implementation evidence.

| Capability domain | v0.5.0 status | Principal evidence |
|---|---|---|
| Foundations and scope | Complete for draft | Foundations, core specification, non-goals, terminology |
| Governance and authority | Complete for draft | Governance model, decision rights, delegation and lifecycle guidance |
| Reference and information architecture | Complete for draft | Architecture views, components, services, interactions, entities and workflows |
| Security architecture | Complete for draft | Objectives, assets, boundaries, threats, adversaries and controls |
| Risk, assurance and resilience | Complete for draft | Risk method, evidence model, evaluation, continuous assurance and resilience |
| Data security and privacy | Complete for draft | Data lifecycle, cryptographic protection, privacy architecture, threats and controls |
| Rights, challenge and remedy | Complete for draft | Affected-party model, transparency, appeal, remedy and accountability |
| Interoperability and profiles | Complete for draft | Profile method, standards register, crosswalks and jurisdiction guidance |
| Reference implementation assets | Complete for draft | Worked scenario, fixtures, schemas and machine-readable catalogues |
| Conformance and implementation guidance | Complete for draft | Scoped claims, assessment guidance and traceability artefacts |
| Documentation and learning experience | Complete | Role-based paths, site map, navigation, diagrams and publication checks |
| Independent deployment evidence | Post-v0.5.0 | Planned for implementation-refinement and candidate-specification phases |

The authoritative limitations and remaining evidence needs are recorded in [Known Limitations](known-limitations.md), [Unresolved Issues](unresolved-issues.md), and [Future Work](future-work.md).
