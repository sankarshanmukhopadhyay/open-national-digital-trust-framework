# Reference maintenance policy

External references are reviewed when a new edition is published, a security or legal advisory affects their use, an interoperability failure is observed, or a profile owner raises a material conflict. Normative dependencies require explicit version selection and migration treatment. Informative references must not silently determine conformance.
