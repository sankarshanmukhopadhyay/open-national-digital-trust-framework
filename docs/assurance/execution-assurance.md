# Execution assurance

Execution assurance evaluates whether a specific action remained within the authorised mandate, applicable policy and runtime constraints. It connects pre-action authority evaluation with observable execution evidence and post-action accountability.

Evidence may include policy decisions, transaction context, tool invocations, approvals, constraint checks, decision receipts and exception records. Profiles must identify which execution events are material, how they are correlated, and how violations trigger containment and remedy.
