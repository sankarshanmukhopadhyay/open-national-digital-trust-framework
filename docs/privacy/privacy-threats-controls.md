# Privacy threats and controls

ONDTF privacy threats include over-collection, secondary use, opaque inference, cross-context correlation, issuer-verifier collusion, status-query surveillance, compelled disclosure, excessive retention, discriminatory profiling, bystander exposure, and unreviewable automated effects.

Controls SHALL be linked to evidence. A policy statement without implementation and operational evidence does not establish privacy assurance.

| Threat | Example controls |
|---|---|
| Cross-context correlation | pairwise identifiers, minimised attributes, query privacy |
| Secondary use | purpose binding, access policy, audit, deletion |
| Excessive retention | event-based schedules, holds governance, destruction evidence |
| Opaque inference | model and rule transparency, explanation, challenge |
| Collusion | separation, independent oversight, unlinkable presentations |
