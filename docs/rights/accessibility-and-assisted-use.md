# Accessibility and assisted use

Rights are ineffective when notice, challenge and remedy channels cannot be used by the people most likely to need them. DTF profiles must address disability access, language, literacy, connectivity, device limitations, assisted use, representation, offline alternatives and protection against coercive intermediaries.

Accessibility evidence should include tested user journeys, supported formats, response times, assistance safeguards and unresolved barriers. Digital-only processes require a documented justification and a safe alternative where exclusion could create material harm.
