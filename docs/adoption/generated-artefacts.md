# Generated artefacts

A completed guided flow produces a profile package rather than one monolithic document.

```text
dtf-profile/
  README.md
  scope-and-purpose.md
  governance-model.md
  institutional-roles.md
  participant-obligations.md
  provider-lifecycle.md
  assurance-model.md
  conformance-model.md
  registry-and-status-model.md
  privacy-rights-and-remedy.md
  recognition-and-interoperability.md
  maintenance-and-change-control.md
  unresolved-decisions.md
  model/
    profile.yaml
    roles.yaml
    requirements.yaml
    assurance.yaml
    conformance.yaml
    dependencies.yaml
    risk-register.yaml
    decision-register.yaml
    review-register.yaml
```

Every generated section must expose its status, source decisions, inherited requirements, unresolved matters, and required reviews. Generation does not convert a provisional answer into an approved governance decision.
