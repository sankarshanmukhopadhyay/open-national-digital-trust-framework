# Worked reference scenario

This section instantiates the ONDTF architecture through a fictional but realistic delegated eligibility decision followed by challenge and remedy. It demonstrates how mandate, authority, delegation, evidence, registry status, policy, decision, effect, receipt, audit, challenge, and remedy fit together.

The scenario is illustrative. It does not prescribe a credential format, identifier method, registry technology, or jurisdictional legal conclusion.
