# Financial Services Profile

Priority use cases include reusable KYC, legal-entity credentials, professional authority, account mandates, beneficial-ownership attestations, sanctions-screening evidence and delegated payment authority.

A reusable KYC credential MUST NOT be treated as a perpetual compliance passport. Regulated entities remain responsible for risk-based decisions, freshness, sanctions checks and suspicious-activity obligations.

High-value delegated transactions SHOULD require proof of current authority, purpose and transaction limits in addition to proof of identity.
