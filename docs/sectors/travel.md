# Travel, Transport and Hospitality Profile

The profile supports domestic and international travel, driver credentials, passenger admissibility, tickets, hotel registration and access control.

It SHOULD support W3C VC and ISO mdoc formats, OpenID4VP-compatible presentation and ICAO-aligned travel credentials where applicable. Biometric processing MUST be purpose-limited, independently assessed and accompanied by non-biometric alternatives where feasible.
