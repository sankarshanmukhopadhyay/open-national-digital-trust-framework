# Travel, Transport and Hospitality Profile

The profile supports domestic and international travel, driver credentials, passenger admissibility, tickets, hotel registration and access control.

It SHOULD support [W3C VC](../standards/references.md#ref-W3C-VC-DM-2.0) and [ISO mdoc](../standards/references.md#ref-ISO-IEC-18013-5-2021) formats, [OpenID4VP](../standards/references.md#ref-OIDF-OID4VP-1.0)-compatible presentation and [ICAO-aligned](../standards/references.md#ref-ICAO-DOC-9303) travel credentials where applicable. Biometric processing MUST be purpose-limited, independently assessed and accompanied by non-biometric alternatives where feasible.
