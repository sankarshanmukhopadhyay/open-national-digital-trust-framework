# Education and Skills Profile

Use cases include degrees, school records, micro-credentials, apprenticeships, professional development and recognition of prior learning.

Profiles SHOULD support verification without issuer contact, status updates for withdrawn qualifications, multilingual semantic mappings and international recognition. A verifier SHOULD request only the qualification evidence necessary for the stated purpose.
