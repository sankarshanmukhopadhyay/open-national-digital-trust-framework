# Core Entities

| Entity | Definition | Canonical ownership |
|---|---|---|
| Jurisdiction | Legal or administrative domain in which a profile applies | ONDTF |
| Sector | Domain of regulated or coordinated activity | ONDTF |
| Trust scheme | Governed arrangement defining participation, policies, services, and assurance | ONDTF, informed by TSMM |
| Governance authority | Entity authorised to set or approve scheme governance | ONDTF |
| Accountable authority | Entity ultimately answerable for a decision, service, or effect | ONDTF/TSMM profile |
| Operating authority | Entity authorised to execute a governed function | ONDTF/TSMM profile |
| Assurance authority | Entity authorised to assess or attest control satisfaction | ONDTF |
| Participant | Entity admitted to a trust scheme in one or more roles | TSMM profile |
| Affected party | Entity whose rights, interests, or conditions may be changed by an admitted effect | ONDTF extension |
| Authority grant | Bounded source of permission, duty, or power | TSMM profile |
| Delegation | Derived authority constrained by scope, purpose, time, and lineage | TSMM |
| Policy | Rule set governing eligibility, duty, prohibition, evidence, or effect admission | TSMM profile |
| Evidence | Verifiable information evaluated in a trust decision | TSMM/TIS |
| Credential | Portable claim container issued under stated authority | External standards/TIS profile |
| Registry record | Published trust artefact used for discovery or resolution | TIS |
| Trust decision | Evaluation determining whether a governed effect may be admitted | TSMM |
| Decision receipt | Durable record of decision basis and result | TIS, semantically aligned to TSMM |
| Admitted effect | Consequence permitted to occur after evaluation | TSMM/ONDTF |
| Incident | Event that compromises or threatens trusted operation | ONDTF |
| Challenge | Formal contest of a decision, record, action, or effect | ONDTF |
| Remedy | Corrective, compensatory, restorative, or preventive response | ONDTF |
