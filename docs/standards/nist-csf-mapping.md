# Informative mapping to NIST Cybersecurity Framework 2.0

This mapping is directional and outcome-oriented. It does not claim that ONDTF adoption constitutes NIST CSF implementation.

| NIST CSF function | Principal ONDTF coverage |
|---|---|
| Govern | governance, authority, risk acceptance, standards policy and oversight |
| Identify | assets, boundaries, dependencies, threats and risk assessment |
| Protect | control framework, authority validation, evidence integrity and access controls |
| Detect | continuous assurance, monitoring, status change and anomaly detection |
| Respond | incident workflow, escalation, communications, containment and challenge |
| Recover | resilience, reconciliation, restoration, remedy and reassessment |

The detailed machine-readable crosswalk is `model/references/nist-csf-2.0-crosswalk.yaml`.
